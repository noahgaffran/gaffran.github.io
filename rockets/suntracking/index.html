The project
The goal of this project is to build a model rocket that uses the position of the sun in the sky to actively stabilize its flight using canards. Based on a 1988 project by George Gassoway
Electronics
The guidance system uses photoresistors to find the location of the sun relative to the rocket's flight path. An arduino nano controls the servos to steer the fins accordingly. This is my first rocketry project that uses custom electronics and it has been a great learning experience so far.
